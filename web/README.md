# Bidstreet Web

## Getting Started

0. Change the repository:
   ```
   cd bidstreet/web
   ```
1. Install the dependencies:
   ```
   yarn
   ```
   or if you use npm:
   ```
   npm install
   ```
2. Create `.env` by following `.env.example`

3. Run the project
    ```
    yarn start
    ```
    or if you use npm:
    ```
    npm start
    ```
