# Bidstreet Web

## Getting Started

1. Change the repository:
   ```
   cd bidstreet/web
   ```
2. Install the dependencies:
   ```
   yarn
   ```
   or if you use npm:
   ```
   npm install
   ```
3. Create `.env` by following `.env.example`

4. Run the project
    ```
    yarn start
    ```
    or if you use npm:
    ```
    npm start
    ```
